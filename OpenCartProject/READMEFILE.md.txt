User Journey :

Creating Registration Page : 
User must have the valid account . For that user need to create the account . User enter the valid credentials i.e Username , Password , Valid Email Id , Select Country .

Login to OpenCart Account : 
User must Login to OpenCart Account with valid credentials . And after login , user can see the  Account Page . Also , validate the Account Page Title

Click on MarketPlace : 
After Logging into the account , user must be able to click on Marketplace link & Selet the product . 

User should be able to Add to Cart : 
After click on MarketPlace link . User should select any product . After selecting product , user should be able to add the product to cart & bclick on Buy option .
